# Day 01 – Touchless LED Switch

## 🎯 Objective
Control an LED with a hand wave using an ultrasonic sensor and Arduino.

## 🔧 Components Used
- Arduino uno or Nano
- HC-SR04 Ultrasonic Sensor
- LED
- 220Ω Resistor
- Jumper Wires
- Breadboard

## ⚡ Circuit Connections
- VCC → 5V
- GND → GND
- TRIG → D3
- ECHO → D4
- LED → D2 (with resistor to GND)
